# EcoPower Tracker

**Version:** 2.1f  
**Author:** Saqib Jawaid  
**License:** GPL 3 or later

## Description

EcoPower Tracker is a WordPress plugin designed to manage and display renewable energy project data. It provides features to track total power generation and CO2 offset, along with detailed project management and visualization capabilities.

## Features

- Manage renewable energy projects (Wind and Solar).
- Track total power generated and CO2 offset.
- Display project details in various customizable formats using shortcodes.
- Import and export project data via CSV files.
- Customizable widgets to show real-time data.
- Support for multiple languages through localization.
- Customizable reporting intervals.

## Installation

1. Download the `ecopower-tracker.zip` file from the GitHub repository.
2. Go to your WordPress admin panel.
3. Navigate to Plugins > Add New > Upload Plugin.
4. Select the `ecopower-tracker.zip` file and click "Install Now".
5. Once installed, activate the plugin.

## Usage

### Shortcodes

EcoPower Tracker offers a variety of shortcodes to display project data in different formats.

#### Total Power Generated

- **Shortcode:** `[ecopower_tracker_total_power]`
- **Description:** Displays the total power generated by all projects.
- **Example:** `Total Power Generated: 12345.67 MWh`

#### Total CO2 Offset

- **Shortcode:** `[ecopower_tracker_total_co2]`
- **Description:** Displays the total CO2 offset by all projects.
- **Example:** `Total CO2 Offset: 10493.82 kg`

#### All Projects Table

- **Shortcode:** `[ecopower_tracker_projects]`
- **Description:** Displays a table listing all projects with their details.

#### Total Power Generated (Number Only)

- **Shortcode:** `[ecopower_tracker_total_power_number]`
- **Description:** Displays the total power generated as a plain number.
- **Example:** `12345.67`

#### Total CO2 Offset (Number Only)

- **Shortcode:** `[ecopower_tracker_total_co2_number]`
- **Description:** Displays the total CO2 offset as a plain number.
- **Example:** `10493.82`

#### Specific Project Details

- **Shortcode:** `[ecopower_tracker_project project_name="Project Name"]`
- **Description:** Displays details for a specific project.
- **Example:** `[ecopower_tracker_project project_name="Solar Plant 1"]`

#### Grouped by Company

- **Shortcode:** `[ecopower_tracker_company company_name="Company Name"]`
- **Description:** Displays aggregated data for all projects under a specific company.
- **Example:** `[ecopower_tracker_company company_name="Solar Corp"]`

#### Grouped by Location

- **Shortcode:** `[ecopower_tracker_location location="Location Name"]`
- **Description:** Displays aggregated data for all projects at a specific location.
- **Example:** `[ecopower_tracker_location location="Location 1"]`

#### Grouped by Type of Plant

- **Shortcode:** `[ecopower_tracker_type plant_type="Plant Type"]`
- **Description:** Displays aggregated data for all projects of a specific type.
- **Example:** `[ecopower_tracker_type plant_type="Solar"]`

#### Total Generation Capacity

- **Shortcode:** `[ecopower_tracker_total_capacity]`
- **Description:** Displays the total generation capacity of all projects combined.
- **Example:** `1500.00 KW`

#### Generation Capacity for a Specific Project

- **Shortcode:** `[ecopower_tracker_project_capacity project_name="Project Name"]`
- **Description:** Displays the generation capacity for a specified project.
- **Example:** `[ecopower_tracker_project_capacity project_name="Solar Plant 1"]`

#### Total Generation Capacity Grouped by Company

- **Shortcode:** `[ecopower_tracker_company_capacity company_name="Company Name"]`
- **Description:** Displays the total generation capacity for all projects under a specified company.
- **Example:** `[ecopower_tracker_company_capacity company_name="Solar Corp"]`

#### Total Generation Capacity Grouped by Location

- **Shortcode:** `[ecopower_tracker_location_capacity location="Location Name"]`
- **Description:** Displays the total generation capacity for all projects at a specified location.
- **Example:** `[ecopower_tracker_location_capacity location="Location 1"]`

#### Total Generation Capacity Grouped by Type of Plant

- **Shortcode:** `[ecopower_tracker_type_capacity plant_type="Plant Type"]`
- **Description:** Displays the total generation capacity for all projects of a specified type.
- **Example:** `[ecopower_tracker_type_capacity plant_type="Solar"]`

### Import/Export CSV

To import or export project data:

1. Go to the "Import/Export" submenu under "EcoPower Tracker".
2. Follow the instructions to upload a CSV file for importing data or download the existing project data as a CSV file.

#### Sample CSV Format

The CSV file should be in the following format with comma-separated values. Ensure the date format is `YYYY-MM-DD`.

| Project Company | Project Name     | Project Location | Type of Plant | Project CUF | Generation Capacity (KW) | Date of Activation |
|-----------------|------------------|------------------|---------------|-------------|--------------------------|--------------------|
| Solar Corp      | Solar Plant 1    | Location 1       | Solar         | 0.20        | 150                      | 2021-05-01         |
| Wind Inc        | Wind Farm 1      | Location 2       | Wind          | 0.25        | 300                      | 2020-01-15         |
| Renewable Energy| Solar Plant 2    | Location 3       | Solar         | 0.18        | 500                      | 2019-08-20         |

### Customizable Widgets

EcoPower Tracker provides widgets to display real-time data on your website. You can configure the appearance and behavior of these widgets from the WordPress Widgets panel.

### Localization

EcoPower Tracker supports multiple languages. Use WordPress's translation functions to translate the plugin's text into your desired language.
