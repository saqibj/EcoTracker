# Overview
EcoPower Tracker is a WordPress plugin designed to track and display renewable energy project data, including power generation and CO2 offset calculations. It helps organizations manage and showcase their renewable energy projects with detailed metrics and visualizations.

# Core Features
- **Project Management**: Track renewable energy projects with details like project number, company, location, and plant type.
- **Power Generation Tracking**: Monitor generation capacity and capacity utilization factor (CUF).
- **CO2 Offset Calculation**: Automatically calculate CO2 offset based on power generation data.
- **Data Import/Export**: Support for CSV import/export of project data.
- **Admin Dashboard**: Comprehensive interface for managing projects and viewing analytics.
- **Frontend Display**: Shortcodes to display project data on the frontend.

# User Experience
- **Admin Users**: Can manage projects, import/export data, and view analytics.
- **Website Visitors**: Can view project information and environmental impact through frontend displays.

# Technical Architecture
- **Backend**: PHP 7.4+, WordPress 5.0+
- **Database**: Custom table `wp_ecopower_tracker_projects`
- **Frontend**: jQuery, responsive design
- **Security**: Nonce verification, input sanitization, capability checks
